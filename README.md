# E-commerce Data Analysis 🛒

## 📌 Project Overview

This project analyzes e-commerce sales data to understand customer purchasing behavior, product performance, and sales trends.

## 🎯 Objectives

* Analyze sales trends over time
* Identify top-selling products
* Understand customer purchase patterns
* Compare categories performance

## 🛠️ Tools & Technologies

* Python
* Pandas
* Matplotlib / Seaborn

## 📊 Key Insights

* Highest sales observed during specific months
* Certain product categories dominate revenue
* Repeat customers contribute significantly to sales
